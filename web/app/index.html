El siguiente paso para una sociedad

con cultura de transacciones electrónicas

Dinérico facilita la integración a la red de dinero electrónico para usuarios MACRO AGENTES y PERSONAS NATURALES.

La herramienta consiste en una aplicación móvil y con soporte web, de fácil uso, que permite realizar diversos tipos de transacciones de manera ágil y segura.

Todos los movimientos de dinero cuentan con las seguridades necesarias gracias al respaldo del Banco Central del Ecuador.

Con la ayuda de Dinérico, el BCE busca aumentar las oportunidades para redes de negocios con intereses de operar como MACRO AGENTES, integrándolos de manera controlada y sustentable.

Impacto Positivo
  • Se generan actividades comerciales personalizadas entre MACRO AGENTES, promoviendo el comercio electrónico.
  • Aumento de la oferta de servicios transaccionales de valor agregado como facturación electrónica, pago de servicios y botón de pago.

Macro Agente

Depósitos

Retiros

Envíos

Gestiona retiros de dinero en efectivo de clientes y recibe depósitos de dinero electrónico de otras cuentas de usuarios que manejan el servicio.
